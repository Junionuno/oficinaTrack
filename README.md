# OficinaTrack - Projeto Prático de Java
Este repositório contém um projeto simples desenvolvido em Java com o objetivo exclusivo de prática e estudo dos conceitos ensinados nas aulas do curso Java Virado no Jiraya do DevDojo.

O foco aqui foi juntar várias peças soltas do aprendizado de backend em um único cenário fictício (um sistema de gerenciamento de ordens de serviço de uma oficina mecânica) para entender como elas funcionam juntas no mundo real.

O código foi estruturado para praticar os seguintes tópicos da linguagem:

Orientação a Objetos Básica: Criação de classes, encapsulamento (atributos privados, getters e setters) e construtores.

Coleções (List e ArrayList): Manipulação dinâmica de dados na memória.

Ordenação e Busca Binária: Implementação de contratos de comparação (Comparable) e uso do método Collections.binarySearch() para buscas eficientes.

Tratamento de Exceções & Regex: Criação de uma exceção customizada e uso de expressões regulares para validar formatos de texto (placas de veículos).

Manipulação de Arquivos (I/O): Leitura e escrita de dados em arquivos .txt usando BufferedReader e BufferedWriter com Try-with-resources.

Conversão de Dados: Passagem de listas dinâmicas para arrays puros (.toArray()).
